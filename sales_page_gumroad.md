# AndroidBoost Pro v1.0

Make your Android phone feel brand new again.

AndroidBoost Pro is a Termux-based optimization toolkit for power users and technicians. It safely removes bloat, boosts performance, improves battery life, and enables a snappy Gaming Mode.

## What you get

- `androidboost_pro.sh` (full Pro script)
- `androidboost_lite.sh` (Lite version)
- Setup instructions
- Basic usage guide

## Features

- Detect and disable common 3rd-party bloat (Facebook, Netflix, OEM stores, etc.)
- One-tap performance + battery tweaks
- Gaming Mode (zero animations, max responsiveness)
- Revert menu to restore everything
- Logs every change so you stay in control

## Requirements

- Android phone
- Termux installed
- Root (Magisk, etc.) for full features

- Rooted phone owners
- Android tinkerers and modders
- Phone repair shops
- Freelancers offering "phone optimization" services
